# Architecture Documentation

This directory contains architectural documentation for the PourStock platform.

## Contents

| Directory | Purpose |
|-----------|---------|
| `adr/` | Architectural Decision Records — formal records of significant technical decisions |
| `history/` | Chronological platform evolution and historical context |

## Usage

- Consult ADRs before proposing changes to established patterns
- Reference `history/platform-evolution.md` for understanding how the system reached its current state
- New architectural decisions should be recorded as ADR documents following the template in `adr/ADR-template.md`
